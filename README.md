# React Slider Component

Компонент слайдера, реализованный на **React** без сторонних библиотек.  
Проект создан как самостоятельный UI-компонент и может использоваться в любых React-проектах для создания галерей, каруселей и презентаций.

---

## Возможности

- React + чистый JavaScript (ES6+)
- Без дополнительных зависимостей
- Автоплей с паузой при наведении
- Touch-события для мобильных устройств
- Клавиатурная навигация
- Индикаторы (dots) с прямым переходом
- Бесконечная прокрутка (опционально)
- Счётчик слайдов
- Оптимизация производительности (useCallback)

---

## Структура слайда

| Поле | Тип | Описание |
|------|-----|----------|
| id | number/string | Уникальный идентификатор |
| image | string | URL изображения (обязательно) |
| title | string | Заголовок слайда (опционально) |
| description | string | Описание слайда (опционально) |

---

## Props компонента

| Prop | Тип | По умолчанию | Описание |
|------|-----|--------------|----------|
| slides | Array | [] | Массив объектов слайдов |
| autoplayDelay | Number | 3000 | Задержка автоплея (мс), 0 = выключен |
| showDots | Boolean | true | Показывать индикаторы (dots) |
| showArrows | Boolean | true | Показывать стрелки навигации |
| infinite | Boolean | true | Бесконечная прокрутка слайдов |

---

## Технические детали

### React хуки
- `useState` — управление состоянием
- `useEffect` — автоплей, клавиатурная навигация, очистка
- `useCallback` — мемоизация функций для оптимизации
- `useRef` — хранение координат touch, ссылки на таймеры

### Оптимизация
- Функции обёрнуты в `useCallback` для предотвращения лишних рендеров
- Минимальное количество re-renders дочерних компонентов
